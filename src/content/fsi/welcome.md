Welcome! This section works exactly like the main blog and the English section,
but it is dedicated to the **Financial Services Industry (FSI)** — payments,
banking, capital markets, insurance, regulation, and related topics.

## How to add a post

Create a new `.md` (or `.mdx`) file inside `src/content/fsi/` with this
frontmatter:

```yaml
---
title: 'Your title'
description: 'A short description'
pubDate: 'Jul 11 2026'
heroImage: '../../assets/blog-placeholder-1.jpg' # optional
---
```

Everything below the frontmatter is the post body, written in Markdown. Once the
file is committed and pushed to `main`, it publishes automatically.

## Ideas for content

- **Payments** — schemes, rails, value-added services
- **Banking & lending** — products, risk, operations
- **Capital markets** — trading, settlement, market structure
- **Regulation & compliance** — KYC/AML, reporting, standards
- **Technology** — core banking, cloud in FSI, data platforms

Happy writing!
