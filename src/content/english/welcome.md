Welcome! This section works exactly like the main blog, but it is dedicated to
English learning — vocabulary, grammar notes, phrases, and bilingual practice.

## How to add a post

Create a new `.md` (or `.mdx`) file inside `src/content/english/` with this
frontmatter:

```yaml
---
title: 'Your title'
description: 'A short description'
pubDate: 'Jul 11 2026'
heroImage: '../../assets/blog-placeholder-1.jpg' # optional
---
```

Everything below the frontmatter is the post body, written in Markdown. Once the
file is committed and pushed to `main`, it publishes automatically.

## Ideas for content

- **Vocabulary** — new words with example sentences
- **Grammar notes** — explanations of tricky rules
- **Phrases & idioms** — with meaning and usage
- **Bilingual practice** — English ↔ Vietnamese translations

Happy learning!
